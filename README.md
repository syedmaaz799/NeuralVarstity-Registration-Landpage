# NeuralVarsity — Agentic AI Career Masterclass

A premium, cinematic landing & registration experience for the NeuralVarsity Agentic AI Masterclass.

Designed with the same restraint as Apple event pages, Stripe Sessions, Linear, Vercel and Framer — engineered as a single Next.js application with intentional motion, premium typography and ambient 3D atmosphere.

## Stack

- **Next.js 14** (App Router)
- **React 18 + TypeScript**
- **TailwindCSS** (custom design tokens)
- **Framer Motion** (UI motion, reveals, micro-interactions)
- **GSAP + ScrollTrigger** (cinematic scroll storytelling)
- **Lenis** (premium smooth scrolling)
- **React Three Fiber / Three.js** (ambient neural network background)

## Getting started

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000).

## Design tokens

```
background:       #0B0B0F
surface:          #111116
elevated surface: #17171D
text primary:     #FFFFFF
text secondary:   #A1A1AA
accent:           #4DA3FF
```

A single accent color is used across the entire experience. No rainbow gradients, no neon overload, no cyberpunk.

## Brand wordmark rule

`NeuralVarsity` is always rendered in **Poppins 800**, solid white, no gradient, no glow.
See `components/brand/wordmark.tsx`.
